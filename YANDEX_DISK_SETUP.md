# Настройка Яндекс Диск для хранения файлов

## 1. Регистрация приложения

1. Перейдите на [https://oauth.yandex.ru/](https://oauth.yandex.ru/)
2. Войдите в свой Яндекс аккаунт
3. Нажмите "Создать приложение"
4. Заполните форму:
   - **Название**: Stwarehouse
   - **Платформы**: Веб-сервисы
   - **Callback URL**: `https://stwarehouse.vercel.app`
   - **Права доступа**: 
     - Доступ к Яндекс.Диску для приложений
     - Чтение всего Диска
     - Запись в любом месте на Диске

5. Сохраните и скопируйте **Client ID**

## 2. Настройка в приложении

Добавьте Client ID в `.env` файл:

```env
VITE_YANDEX_CLIENT_ID=your_client_id_here
```

Или используйте напрямую в компоненте:

```tsx
<YandexDiskFileManager 
  clientId="YOUR_CLIENT_ID"
  redirectUri="https://stwarehouse.vercel.app"
  basePath="/stwarehouse"
/>
```

## 3. Использование

Добавьте компонент в нужное место приложения, например в раздел документов или настроек.

### Возможности:
- ✅ Загрузка файлов на Яндекс Диск
- ✅ Скачивание файлов
- ✅ Создание папок
- ✅ Удаление файлов
- ✅ Получение публичных ссылок
- ✅ Прогресс загрузки

## 4. Безопасность

- Токен хранится в localStorage
- Для production рекомендуется использовать backend-proxy
- Не коммитьте Client ID в публичный репозиторий

## Альтернатива: Backend Proxy

Для лучшей безопасности можно создать backend-прокси:

```javascript
// supabase/functions/yandex-disk/index.ts
// Храните токен в Supabase и делайте запросы через Edge Functions
```

Это скроет токен от клиента и даст больше контроля.
